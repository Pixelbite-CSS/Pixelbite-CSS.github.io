How it works?

Image is from unsplash.com
If you have been ever wondering how this work, let me explain.
Basically you have a CSS library, which can work on its own, but for more features, you'll need to use JavaScript library too. When you have an element, you will add a class (for example bg-red50) 
<h3 class="bg-red50">Hello world</h3>

Hello world

Of course this library has way more features and classes than background-color. Please refer to our docs.
