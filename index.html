FAQ's

What is PixelbiteCSS?
What is PixelbiteCSS?
PixelbiteCSS is a framework, that uses customizable CSS and Javascript to help create your project faster. You can use PixelbiteCSS to create custom components, or use already made ones in the Javascript.
How can I join?
How can I join?
You can join our team by visiting our Github group on https://github.com/Pixelbite-CSS. Framework is open-source so everybody can contribute without joining.
Have any questions? Ask us in our Github issues.
