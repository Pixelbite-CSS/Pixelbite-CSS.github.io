Framework that makes website development easier.

Customisable CSS and Javascript framework, that uses its own classes like flex, ta-center or slideshow to help you create websites faster and responsive.

Getting started

Start using PixelbiteCSS now by selecting version you want to use or you can optionally donwload library via npm or Github.

Read installation docs

Easy to customize

You can customize every aspect of PixelbiteCSS's framework. Change or add colors, variables, classes, elements, scripts and more with fully customizable .conf files.

Custom components

Since version 1.5 you can use your own custom created components stright inside of HTML files. Use tags such as ${variable} inside your code to make use of custom variables.

Custom themes

You can use, create or edit themes on Github themes repository to make websites even faster and nicer.

Darkmode

PixelbiteCSS uses it's own custom darkmode. Implement your own custom darkmode theme for your website with easy to use prefixes.

FontAwesome support

PixelbiteCSS supports implementation of FontAwesome's icons. Change FontAwesome url in config file.
